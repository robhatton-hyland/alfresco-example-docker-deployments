# alfresco-example-docker-deployments
A collection of example Alfresco docker deployments

This collection is is designed to help Alfresco users and technical support analysts build or get inspiration on how to build, specific environments or stacks in Docker, for both ACS and APS. The environments are ones that I've either built as part of my role as a technical support analyst or environments I've set up to test specific situations. I aim to make as much use of the .env file as possible, to allow for different versions to be used where needed.
